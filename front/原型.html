# 原型

他：你可以走了，我知道你什么都不懂。 我：我觉得我还可以抢救一下。 他：那你告诉我原型是什么。 我：出口在哪里？

# 创建对象

  1. 字面量

    var obj = {x:1};

  2. new

    var obj = new Object({x:1});
// or
var O = function(){this.x=1;};
var obj = new O();

  3. Object.create()

    var P = {x:1};
var obj = Object.create(P);
// obj.__proto__ === P

# 原型关系

  1. 每个对象都有__proto__(函数也是对象)
  2. 只有函数默认有prototype，对象没有

# instanceof

instanceof关联的是构造函数的prototype，而不是构造函数本身
另外，只要实例的__proto__在prototype链中，instanceof就会返回true
举个栗子：
设一个构造函数F，它的实例为f，那么

// 结果
f instanceof F -> true
f instanceof Object -> true
// 原因
f.__proto__ === F.prototype
F.prototype.__proto__ === Object.prototype

因此如果想判断实例的构造函数，可以用constructor(当然这货可能会被改，详情见下一章)

# new运算符

运行流程：

  1. 新建对象，赋值其__proto__为foo.prototype
  2. 构造函数foo传入参数并执行，上下文this被指定为这个新对象
  3. 如果构造函数返回了一个对象，那么这个对象会取代整个new的结果，否则返回new出来的新对象

运行模拟：

function new(func) {
  var o = Object.create(func.prototype);
  var k = func.call(o);
  if(typeof k === 'object') {
    return k;
  } else {
    return o;
  }
}
